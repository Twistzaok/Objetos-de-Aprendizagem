Intents
O que são?
Uma Intent é um objeto que é utilizado para solicitar uma ação de outro componente do aplicativo. Embora possam trabalhar de diversas formas, possui 3 formas fundamentais, iniciando uma atividade, um serviço ou fornecendo uma transmissão.

Uma Intent pode ainda ser classificada como Implícita ou Explícita, sendo que:

Explícita: especificam qual aplicativo atenderá ao Intent, fornecendo dados de um componente qualificado. Geralmente, é utilizado quando sabe-se o nome da atividade ou serviço que se iniciará.

Implícita: não especificam qual aplicativo atenderá ao Intent, declarando apenas qual ação deverá ser executada.
Iniciando
Como podem trabalhar de diversas formas, existem várias formas de iniciar uma Intent, sendo as principais:

Activity: É possível iniciar uma nova instância de Atividade passando um Intent para startActivity(). O Intent descreverá a Activity e carregará os dados necessários.

Services: É possível iniciar um Serviço utilizando um JobScheduler. É possível também vincular um Serviço em outro componente passando um Intent para bindService().

Services: É possível fornecer uma transmissão para outros aplicativos passando um Intent ao sendBroadcast() ou sendOrderedBroadcast().
Criando
Um objeto Intent carrega informações para determinar o componente que será iniciado, portanto, ao criar um Intent pode ser necessário fornecer alguns dados, como por exemplo:

Nome: Opcional, se for definido tornará a Intent explícita. É possível definir o nome do componente com setComponent(), setClass(), setClassName() ou com o construtor Intent.

Ação: String que especifica a ação a se realizar. É possível especificar a ação para um intent com setAction() ou com um construtor Intent.

Dados: É um objeto URI que referencia os dados a serem aproveitados. Para definir só um URI de dados, utilize setData().

Categoria: É possível especificar uma categoria com addCategory().
